= eselling

A Java desktop app for maintaining a list of items for sale, and exporting them to various markets
such as eBay, Kijiji, etc.

The model code is from my previous human-coded eselling web app.
Much of the Java code was written by Claude.ai.

== HowTo Use

In `src/main/resources/META-INF`, copy the sample file to `persistence.xml`, and modify
it to suit. You'll need at least a permanent database (PostgreSQL recommended).
